<section class="installation">

## Installation

```bash
npm install @stdlib/constants-float64-phi
```

</section>

<section class="usage">

## Usage

```javascript
var PHI = require( '@stdlib/constants-float64-phi' );
```

#### PHI

The [golden ratio][phi-value].

```javascript
var bool = ( PHI === 1.618033988749895 );
// returns true
```

</section>
